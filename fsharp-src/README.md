# AcademicRoadmapCore F# Project

This folder contains the F#/.NET 8 implementation of the core Academic Roadmap logic.

It models degree tracks, difficulty levels, weekly study plans, task progress, GPA calculation, recommendations, and ICS calendar export data. The GitHub Pages UI in the repository root remains the live client-only demo, while this project documents and verifies the F# application logic.

## Build

```bash
dotnet build AcademicRoadmapCore.fsproj
```

## Run

```bash
dotnet run --project AcademicRoadmapCore.fsproj
```
